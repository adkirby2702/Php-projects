# custom-project-adkirby2702
This is a project that reads details about record collection from a text file. 
In the recordLists page, the user can click on a record to be sent to the recordDetail page for the record, which has facts about the record in the collection.
The number of records clicked are stored in a session variable along with a session variable hat tracks the ones clicked.
The user can also add to the txt file using the recordAdd page. This is found at the link at the bottom of the recordList page
